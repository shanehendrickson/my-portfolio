About Me


I'm a self taught web developer who loves learning and being challenged.
I spent twenty years working in shipping and receiving. While I enjoyed the work, I eventually realized it was getting me nowhere. There was no upward momentum and nothing new to learn or discover.
So I quit.


Now I'm learning as much as I can about web and software development. I started with basic HTML and CSS and have moved into JavaScript its associated technologies. I am woking my way toward C# (and beyond) so that I can dig deeper into Unity game development (primarily with Titan Realm, a hobby indie game studio I co-founded.)


When I'm not working on projects, you can find me playing board games, supporting fellow indie video game developers, or watching (and talking about) horror movies.
